# Cursor 专属配置
## Cursor 使用规范
- 使用 Composer（Cmd+I）进行多文件编辑
- 使用 @file 引用具体文件上下文
- 使用 @docs 引用官方文档
- 改动前先用 Cmd+K 快速确认方案
## Cursor Rules 特定指令
- 修改文件前，先输出将要修改的文件列表
- 每次只做一件事，不要一次修改太多文件
- 生成代码后检查是否有 TypeScript 错误
- 优先复用已有组件，不新建重复组件
## 代码生成偏好
- 生成完整可运行的代码，不省略导入
- 新建文件时包含完整的类型定义
- 涉及环境变量时，同步更新 .env.example
